# 商品App

baseUrl：http://localhost:8080

### 1、商品列表

/product/list

参数:  无

返回:

```
{
    list:[
       {
       	 wallImage:[String],
       	 title:[String],
       	 productPrice:[Float],
       	 productId:[Int]
       }
    ]
}
```





### 2、商品详情

/product/info

参数：

productId:[Int]

返回：

```
{
	coverImages:[String[]],
	title:[String],
	productPrice:[Float],
	productDetail:[String]
}
```

# SQL DDL

```sql
  CREATE TABLE `product`(

  `id` BIGINT UNSIGNED NOT NULL AUTO_INCREMENT COMMENT '商品ID',
  
  `cover_images` VARCHAR(2000) CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci NOT NULL COMMENT '封面图，用$拼接',
  
  `title` VARCHAR(255) CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci NOT NULL COMMENT '商品标题',
  
  `product_price` DECIMAL(10,2) NOT NULL COMMENT '商品价格',  
  
  `product_detail` VARCHAR(2000) CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci NOT NULL COMMENT '商品详情',
  
  `create_time` INT UNSIGNED NOT NULL,
  
  `update_time` INT UNSIGNED NOT NULL,
  
  `is_deleted` TINYINT UNSIGNED NOT NULL DEFAULT '0',
  
   PRIMARY KEY (`id`),
   
   KEY `idx_title` (`title`),
   
   KEY `idx_product_price` (product_price)

)ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='商品表'
```
